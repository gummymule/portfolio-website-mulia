# Personal Portfolio Website

A modern dark-themed developer portfolio website built with pure HTML, CSS, and JavaScript.

Live Demo: https://gummymule.github.io/portfolio-website-mulia

---

## Overview

This portfolio showcases:

- Professional experience
- Technical skills
- Featured projects
- Contact information

The design focuses on:
- Minimalist aesthetic
- Smooth animations
- Responsive layout
- Elegant typography
- Clean developer-focused presentation

- HTML5
- CSS3
- Vanilla JavaScript

### Fonts
- DM Sans
